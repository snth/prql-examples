# prql-exec

`prql-exec` is an experimental shell script to make
system tools that accept SQL input easily accessible
as PRQL libraries.

## Getting started

### Install prql-exec

```sh
git clone https://github.com/snth/prql-examples.git
cd prql-examples
source prql-cli.sh
export PRQL_LIB_PATH=".:$(pwd)"
```
